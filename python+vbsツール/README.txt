ツールの構成

①main.py・・・・メインのフロー。ここからGUIでgurafu.pyとYomitori.vbsを呼び出す。
                 GUI画面で①「生成」ボタンを押下するとYomitori.vbsが起動
                 ②真ん中のテキストボックスに文字を入れて「ポップアップ」ボタンを押下するとテキストボックスの文字がポップで出てくる。
                 ③グラフのボタンを押下すると、グラフが表示される。
②gurafu.py・・・ランダムで生成された２つの変数でグラフを作成
③Yomitori.vbs・・「popup.txt」を１行ずつ読み取り、メッセージボックスで表示する。
